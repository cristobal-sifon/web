readfile

Download

readfile is a versatile module for reading ASCII table files of different formats. It allows the specification of which columns will be returned and the format that the contents of each column should have. If the columns are specified, the table does not need to be complete (e.g., it can have comments at the ends of lines) as long as all lines contain the specified columns. It is implemented as a python module, so it can be used within any python program.

Usage:

Put the file in your python path. Then, in a python code,

> import readfile

For help, type

> help(readfile)

The module contains the following functions (links will take you to individual help files):

dict(file)
Creates a python dictionary, using the first line as a header, which contains the names of each column.
header(file)
Reads the header from a file, defined in the first line or as a list of column names.
table(file)
Reads the data from the file and returns a numpy array.
