redsequence

Download

redsequence is a module that allows the user to extract the red sequence from a sample of galaxies, useful for galaxy cluster studies. From a set of galaxy magnitudes and colors (and color uncertainties), redsequence identifies red galaxies using the Error-Corrected Gaussian Mixture Model of Hao et al. (2009, ApJ, 702, 745).

Usage:

First install ecgmmPy, as described here.

Put the file in your python path. Then, in a python code,

> import redsequence

The module to be used is called fit. For help, type

> help(redsequence.fit)

To estimate the best-fit red sequence,

> rs = redsequence(rmag, color, *optional_args)
