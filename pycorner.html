My Python Corner

Python is one of the simplest, most flexible programming language, and the number of available resources is simply gigantic. Having (shortly) been through C and IDL (and IRAF), I don't hesitate when I have to recommend a programming language.

In this page you will find various python codes that I have written throughout the years, that can be useful for different purposes. I usually update them according to my needs but not necessarily on a rigurous, regular basis. They are free to be used, modified and distributed, but I would appreciate being notified about it. The links will take you to help pages where you will be able to download the corresponding code. All codes are provided as python scripts with comments (of course!) and help pages, so they are easy to modify and debug (or you can always contact me about that). Below I've also posted some general python resources.

Most of these packages can now be found in my github profile, where I keep everything up to date. Some of the packages in this page may be slightly outdated but they all do their job, and the help pages here are pretty complete.

lnr
Linear regression in the presence of correlated errors in both variables and intrinsic scatter. Methods implemented are: a python translation of BCES (Akritas & Bershady, 1996), a python wrapper for the widely used IDL MCMC code of Kelly (2007), a maximum likelihood estimator, and a simple MCMC code written with emcee.
plotspec
A simple tool to plot 1d spectra along the corresponding 2d spectrum.
pygmos
A Python/PyRAF pipeline to reduce GMOS spectra. (You can also download a quick guide through the GMOS reduction process.)
readfile
A versatile module for reading ASCII table files.
redsequence
A module to fit the red sequence from a set of galaxy magnitudes and colors using a maximum likelihood approach. Requires the Error-Corrected Gaussian Mixture Model of Hao et al. (2009) to separate blue and red galaxies.
